# 🛡️ Solidity Sentinel

**Solidity Sentinel** is a next-generation smart contract security scanner
built for Ethereum-based Solidity contracts.

It combines **Slither static analysis** with **custom threat detection**
to identify vulnerabilities, centralization risks, and unsafe logic.

## 🚀 Features
- Static analysis using Slither
- Custom threat engine (minting, blacklist, self-destruct, fees)
- Security scoring system
- Clean dark-mode UI
- Line-of-Code (SLOC) analysis
- Professional vulnerability report

## 🧪 Tech Stack
- Python (Flask)
- Slither Analyzer
- Solidity
- HTML / CSS / JavaScript

## 🛠️ Installation
pip install -r requirements.txt
python app.py
